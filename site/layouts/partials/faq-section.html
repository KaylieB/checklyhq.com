Frequently Asked Questions

What is Puppeteer?

Puppeteer is a framework developed by Google's Chrome team. It enables developers to easily run standard Chrome browsers in "headless" mode. Although it doesn't output anything to a screen it keeps all of its features for rendering a web page. To get you started with the Puppeteer framework, we created the GitHub repo checkly/puppeteer-examples with example scripts.

How is it different from PhantomJS or Selenium?

PhantomJS and its partner library CasperJS are based on the QT Webkit browser. Also, they are not "native" Node.js applications. Selenium and the Webdriver standard are Java applications that "talk to" browsers. Setup is complex and you always need Java.
